# ספרדית בלי כאב ראש

הפרויקט כולל את הכריכה, עמודי הפתיחה וכל עשר היחידות של הספר בפורמט HTML.

## פתיחה

פתחו את `00 Cover + Front.dc.html` בדפדפן. תוכן העניינים מקשר לכל היחידות, ובסוף כל יחידה מופיעים כפתורי ניווט.

## קבצים מרכזיים

- `00 Cover + Front.dc.html`: כריכה, הקדמה ותוכן עניינים
- `Unit 01.dc.html` עד `Unit 10.dc.html`: כל יחידות הספר
- `doc-page.js` ו-`support.js`: רכיבי התצוגה וההדפסה
- `manuscript-extracted.txt`: כתב היד המלא
- `build-units.mjs`: בנייה מחדש של יחידות 3 עד 10 מתוך כתב היד

כדי לבנות מחדש את יחידות 3 עד 10 לאחר שינוי בכתב היד, מריצים:

```bash
node build-units.mjs
```
