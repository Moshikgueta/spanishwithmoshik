SPANISH WITH MOSHIK — READY-TO-UPLOAD WEBSITE

This ZIP contains the complete static website:
- Landing page
- Full self-study book
- Separate interactive workbook
- 370 activities with feedback, hints, model answers and saved progress

HOW TO UPLOAD
1. Extract the ZIP file.
2. Upload ALL files and folders inside it to the root of your website.
3. Make sure index.html is in the root, next to the folders assets, book and workbook.
4. Do not upload only the PDF or only one folder.

GITHUB PAGES
1. Extract the ZIP.
2. Upload all extracted items to the root of your GitHub repository.
3. In Settings > Pages, select Deploy from a branch, then main and /(root).
4. Wait a few minutes and refresh the GitHub Pages address.

IMPORTANT
- The workbook works on static hosting and does not need a server or API key.
- Student answers and progress are saved in the browser on the same device.
- Keep the folder names unchanged so all links continue to work.
